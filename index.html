    discoBit - Remote Application Configuration Management - Java Client

    Project maintained by v3rticle-oss Hosted on GitHub Pages — Theme by mattgraham

    discoBit is a service which enables remote configuration of applications. Configurations are read from the central server during bootstrap or on demand by your software.

    About

    Instead of providing each installed application, script or process with local configuration you shift everything to a central instance which serves as a single point of truth. All properties are organized in configuration buckets which are uniquely identified and secured from unauthorized access.

    Your software calls the central service cluster via REST and reads the applied properties when appropriate. You need to provide a single remote-configuration-ID or a unique identifier for the calling node (hostname, IP) and security credentials to access the remote configuration.

    Apply changes to your configs for massive infrastructures without any need for rollouts or restarts.

    See www.discobit.com

    Java Client Library - Usage

    Option 1: Java Agent Configuration

    
java -javaagent:PATHTO/discobit-client-java-0.6.jar -jar yourapplication.jar foo.bar.Application
-Ddiscobit.repository.url=http://discobit.foobar.com:80
-Ddiscobit.repository.username=yourusername
-Ddiscobit.repository.password=yourpassword
-Ddiscobit.cfg.uuid=6992f5aa-c215-4beb-8645-150b9e8c8eab

    Option 2: Programmatic

    {% highlight java %} // get the client DiscobitClient discobit = DiscobitClientFactory.getClient(); // create an application space int spaceID = discobit.createApplicationSpace("foo-space", "john@foobar.com", "John Doe", "http://foobar.com"); // create a configuration within that space UUID cUUID = discobit.createConfiguration(spaceID, "foo-prd-01" , "foo configuration"); // create a property boolean success = discobit.createConfigProperty(cUUID, "test1", "value1"); // read the propery String value = discobit.getConfigProperty(cUUID,"test1", false); {% endhighlight %}
